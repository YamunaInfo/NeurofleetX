Smart City Database Schema (MySQL)
==================================
Modules Covered:
1. Login & Signup (users)
2. Vehicle Booking (vehicles, drivers, bookings)
3. Emergency Management (emergencies)
4. Traffic Signals (traffic_signals)
5. Analytics (analytics)
6. AI Control (ai_control_logs)
7. Digital Twin (digital_twin)
8. User Profile (user_profiles)

Usage:
1. Open MySQL or phpMyAdmin.
2. Run:  SOURCE smart_city_db.sql;
   or:   mysql -u root -p < smart_city_db.sql
3. This creates `smart_city_db` with all required tables and sample data.
